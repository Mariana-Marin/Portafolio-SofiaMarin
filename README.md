## 🛠️ Tecnologías y Características del Desarrollo

### 💻 Stack Tecnológico
*   **Estructura Semántica:** HTML5 con etiquetas claras (`<nav>`, `<section>`, `<footer>`, `<article>`).
*   **Estilos:** CSS3 puro con variables personalizadas (`:root`), flexbox, grid layouts y técnicas de diseño responsivo.
*   **Lógica e Interacción:** Vanilla JavaScript (sin librerías pesadas) para optimizar el rendimiento y garantizar la fluidez.

### ⚡ Características y Efectos Especiales
1.  **Protocolos Desplegables (*Collapsible Steps*):** Cada tarjeta de servicio permite expandir y contraer su paso a paso, manteniendo la interfaz despejada.
2.  **Carrusel Multimagen en Tarjetas:** Los servicios con varias imágenes asociadas (como Dermaplaning o Limpieza Profunda) cuentan con mini-carruseles automáticos e interactivos con indicador numérico (ej. `01 / 04`).
3.  **Animaciones de Entrada (*Scroll Animations*):** Uso de la API nativa de JavaScript `IntersectionObserver` para activar transiciones suaves de opacidad y desplazamiento al hacer scroll en las tarjetas y secciones.
4.  **Optimización Móvil (Mobile-First responsive design):** Menú adaptable a pantallas táctiles, botones adaptados para clics móviles y soporte completo para navegación fluida.
5.  **Optimización de Movimiento Reducido:** Soporte para la media query `@media (prefers-reduced-motion: reduce)`, desactivando animaciones si el usuario lo tiene configurado en su sistema operativo.
